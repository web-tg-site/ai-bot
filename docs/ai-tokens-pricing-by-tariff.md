# AI-кредиты по тарифам: расчёт с фиксированной маржой

## Модель маржинальности

**Базовый тариф:** LITE, 1 месяц — **2 990 ₽**

| Показатель                   | Значение             |
| ---------------------------- | -------------------- |
| Цена                         | 2 990 ₽              |
| Расход на пользователя (API) | **600 ₽** (20,07%)   |
| Чистая прибыль               | **2 390 ₽** (79,93%) |

**Правило для всех тарифов:** то же соотношение расходов и прибыли.

```
расход_₽     = цена_₽ × (600 / 2990)
прибыль_₽    = цена_₽ × (2390 / 2990)
расход_%     = 20,07%
прибыль_%    = 79,93%
```

### Перевод расхода в токены

Курс из базового тарифа: **2 990 ₽ = 37 USDT** → 1 ₽ ≈ $0.012375

```
токены = расход_₽ × (37 / 2990) / 0.002
       = цена_₽ × 600 / 2990 × 37 / 5.98
       ≈ цена_₽ × 1.2416
```

**1 токен = $0.002**

### Стоимость операций (для расчёта «сколько можно сделать»)

| Операция                             | Единица                | Токены |
| ------------------------------------ | ---------------------- | ------ |
| Тестовый запрос (GPT)                | 1 вопрос               | **1**  |
| Изображение (Seedream / Nano Banana) | 1 картинка             | **20** |
| Видео VEO 3.1 Lite                   | **3 секунды**          | **75** |
| Аудио ElevenLabs                     | 1 озвучка (~500 симв.) | **40** |

> Видео: $0.05/сек × 3 сек = $0.15 → 75 токенов.

## Формулы (справочно)

```
расход_₽      = цена_₽ × 600 / 2990
прибыль_₽     = цена_₽ × 2390 / 2990
токены        = floor(цена_₽ × 600 / 2990 × 37 / 0.002 / 2990)
запросы       = токены ÷ 1
изображения   = floor(токены ÷ 20)
видео_3с      = floor(токены ÷ 75)
аудио         = floor(токены ÷ 40)
```
