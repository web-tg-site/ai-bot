# Стоимость генераций (AI-токены)

Одна генерация списывает токены с баланса подписки.  
Ниже — базовые цены и как считается итог.

---

## Как считается цена

**Текст / картинка / озвучка без длительности**

> Итог = базовая цена × множитель качества × множитель разрешения

**Видео и озвучка с длительностью**

> Итог = (базовая цена + цена за секунду × длительность) × множители

Число всегда округляется **вверх** до целых токенов.

### Множители (если доступны в настройках)

| Параметр            | Вариант             | Множитель |
| ------------------- | ------------------- | --------- |
| Разрешение картинки | 1K                  | ×1        |
|                     | 2K                  | ×1,5      |
|                     | 4K                  | ×2        |
| Качество картинки   | auto / low / medium | ×1        |
|                     | high                | ×1,5      |
| Разрешение видео    | 720p                | ×1        |
|                     | 1080p               | ×1,25     |
| Качество видео      | standard            | ×1        |
|                     | high                | ×1,5      |

Если настройка недоступна для инструмента — множитель = ×1.

## Картинки

Базовая цена — при разрешении **1K** и обычном качестве.

| Инструмент  | Базовая цена |
| ----------- | ------------ |
| Flux        | 15           |
| Nano Banana | 20           |
| Seedream    | 20           |
| Midjourney  | 30           |
| GPT Images  | 40           |

**Пример:** Flux 2K + high = 15 × 1,5 × 1,5 = **34** токена.

## Видео

| Инструмент | Цена за секунду | Длительность по умолчанию | Пример на default |
| ---------- | --------------- | ------------------------- | ----------------- |
| Higgsfield | 15              | 5 сек                     | 75                |
| Veo        | 25              | 4 сек                     | 100               |
| HeyGen     | 25              | 5 сек                     | 125               |
| Seedance   | 34              | 5 сек                     | 170               |
| Kling      | 63              | 5 сек                     | 315               |
| Sora       | 150             | 10 сек                    | 1500              |

**Пример:** Veo 8 секунд = 25 × 8 = **200** токенов.  
**Пример:** Sora 10 сек в 1080p = 150 × 10 × 1,25 = **1875** токенов.

### Topaz (улучшение качества видео/фото)

Базовая цена: **40** токенов при масштабе ×2.

| Масштаб | Цена |
| ------- | ---- |
| ×2      | 40   |
| ×4      | 80   |
| ×6      | 120  |

Формула: 40 × (масштаб ÷ 2).
