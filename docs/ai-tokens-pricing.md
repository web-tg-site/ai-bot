# AI-кредиты (токены): расчёт стоимости и тарифов

## Краткое резюме

1. Все генерации переводим в **единую валюту — AI-кредиты (токены)**.
2. Базовый курс: **1 токен = $0.002** (0.2 цента).
3. Токены начисляются **раз в месяц** по типу подписки.
4. Рекомендуемый API-бюджет: **~30% от цены подписки** (остальное — маржа, сторонние API, поддержка).
5. Текущие значения PRO (15 000) и BUSINESS (50 000) **слишком щедрые** — дают 40–67% ухода только на API.

---

## Базовые единицы расчёта

| Параметр            | Значение                           |
| ------------------- | ---------------------------------- |
| 1 токен             | $0.002                             |
| 1 видео             | 5 секунд, 720p                     |
| 1 аудио             | 500 символов (~30 сек речи)        |
| 1 картинка          | 1 изображение ~1024×1024           |
| 1 GPT-запрос        | зависит от типа (см. таблицу ниже) |
| Комиссия OpenRouter | +5.5% при пополнении кредитов      |

**Формула перевода:**

```
токены = округление_вверх(стоимость_в_USD / 0.002)
```

## Рекомендуемые токены на подписку

### Логика

- API-бюджет ≈ **28–30%** от месячной цены подписки.
- Токены начисляются **каждый месяц** (не за весь период сразу).
- Соотношение тарифов: **LITE : PRO : BUSINESS ≈ 1 : 2 : 4**.
