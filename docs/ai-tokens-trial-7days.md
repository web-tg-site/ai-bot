# Пробная подписка на 7 дней — расчёт AI-кредитов

## Условия

| Параметр                      | Значение             |
| ----------------------------- | -------------------- |
| Срок                          | **7 дней**           |
| Себестоимость (расход на API) | **50 ₽** на человека |
| Цена для пользователя         | 0 ₽ (бесплатно)      |
| Курс (из LITE)                | 2 990 ₽ = 37 USDT    |
| 1 токен                       | **$0.002**           |

---

## Сколько это в кредитах

```
расход_USD = 50 × (37 / 2990) = $0.619
токены     = $0.619 / $0.002   = 309.4
```

### **309 AI-кредитов** на 7 дней

> В коде: `test-sub.handler.ts` и `tokens-number-by-sub-name.config.ts` → **309** токенов. Текст бота: **50 / 5 / 1 / 2** (запросы / картинки / видео / аудио).

> Дефолт VEO в UI — **6 сек** (150 токенов), якорь расчёта — **3 сек** (75 токенов).

### Если оставить 3 аудио (как сейчас в тексте)

| Операция      | Лимит  | Токены  |
| ------------- | ------ | ------- |
| AI-запросы    | **50** | 50      |
| Изображения   | **5**  | 100     |
| Видео (3 сек) | **1**  | 75      |
| Аудио         | **3**  | 120     |
| **Итого**     |        | **345** |

Перерасход: **+36 токенов (~5.8 ₽)** сверх бюджета 50 ₽.

Чтобы уложиться с 3 аудио, урезать запросы:

| Операция      | Лимит  | Токены  |
| ------------- | ------ | ------- |
| AI-запросы    | **39** | 39      |
| Изображения   | **5**  | 100     |
| Видео (3 сек) | **1**  | 75      |
| Аудио         | **3**  | 120     |
| **Итого**     |        | **334** |

Нужно **39 запросов + 5 картинок + 1 видео + 3 аудио** или **50 + 5 + 1 + 2 аудио**.
